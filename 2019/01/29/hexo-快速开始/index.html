hexo 快速开始

hexo 是常用的静态网页构建工具，支持markdown，拥有丰富的插件。常用于构建个人博客。

环境

OS window 10 1809

node 10.13.0

npm 6.6.0

PowerShell 5.1.17763.134

hexo 3.8.0

hexo-cli 1.1.0

开始构建

shift+右键 启动 PowerShell，创建 hexo 目录（构建器项目）

进入目录

全局安装 hexo-cli，如果还没有安装nodejs和git，请参考hexo文档，此处不再重复记录

初始化 hexo

安装依赖包

生成静态资源

本地启动sever测试

浏览器访问 localhost:4000

可以看到一个简单的个人博客系统已经完成了！

参考 hexo docs

更换主题

这里以 Next v6.0.x 为例子

下载主题

修改主题，修改./_config.yml中的theme为next

注意区分./themes/next/_config.yml

为了方便区分，下面更换称呼

./_config.yml称为系统配置

./themes/next/_config.yml称为主题配置

保存

清理缓存

重新构建

本地启动

浏览器访问 localhost:4000

成功更换主题！

    定位到 valine

    在主题配置中，找到sidebar

    例如修改显示方式 display

  3. 导航菜单

    在主题配置中，找到menu

    注释tags等，读者可以自由选择

    在根目录执行指令

    增加内容，关闭评论和指定页面类型

    执行后，会创建文件夹./source/tags/，并在里面初始化tags页面的md文件index.md

    重新构建后，可以看到页面多了一个导航栏tags

    填写 AppID 和 AppKey
